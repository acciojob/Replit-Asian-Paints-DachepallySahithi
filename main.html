Replit Asian Paints

The task is to create an application having a 3 X 3 grid numbered as 1,2,3...9 with id as the number assigned to it. For eg : the third element of grid have the id 3 after this the application should also have an input for input-id and one for the color where we can input color as color-id. After we submit this we should observe the background color of that grid to be changed and after we change it to other first every grid element background-color should be set transparent after this the particular ids color is changed.

Instructions

The input for block id should have id block_id and color should have id as colour_id. The Change color button should have an id change_button and a reset button with id Reset.

The grid container should have an id of grid-container and the item should have an id as grid-item.
